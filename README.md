# Salesforce DevOps Center Demo Repository

This repository contains demo projects for showcasing Salesforce DevOps Center capabilities.

## Projects

### 1. Customer Portal Enhancement
Modernizing the customer self-service portal with enhanced case management and knowledge base integration.

### 2. Sales Cloud Optimization
Streamlining sales processes with improved opportunity management and forecasting.

### 3. Service Cloud Modernization
Upgrading service operations with AI-powered case routing and omnichannel support.

### 4. Marketing Automation Suite
Building comprehensive marketing automation with lead scoring and campaign analytics.

## Branch Strategy

- `main` - Production-ready code
- `project/*` - Project-level branches
- `feature/*` - Feature development branches

## DevOps Center Setup

This repo is designed to work with Salesforce DevOps Center for streamlined deployment workflows.
